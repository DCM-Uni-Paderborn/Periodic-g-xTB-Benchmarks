# DMC-ICE13 periodic g-xTB benchmark

This directory contains the Part-I DMC-ICE13 structures, native-Bloch inputs,
reference energies, current frozen-build results, and figure sources. Relative
energies are reported per H2O against ice Ih evaluated on the same k-point
mesh.

## Protocol

For phase `p` and mesh edge `N`,

```text
R_p(N) = E_p(N)/n_p - E_Ih(N)/n_Ih .
```

A phase is converged at the denser endpoint of the first adjacent pair that
satisfies

```text
|R_p(N) - R_p(N-1)| <= 0.10 kJ mol-1 per H2O .
```

There is no aggregate RMS or second consecutive-pair gate. A complete adaptive
MAE is reported only after all twelve non-reference phases have passed. Odd
meshes are unshifted; even MacDonald meshes use the frozen half-grid-compatible
shift in the versioned inputs.

The current Part-I production cap is `8x8x8`.  An unresolved phase at that cap
remains explicitly non-converged but does not create a scheduled `9x9x9`
endpoint.  This keeps the reported `<=8x8x8` statistic provisional rather than
silently treating the cap as a convergence criterion.

## Current data

- `data/dmc_ice13_gxtb_same_build_fixed_mesh_mae.csv`: complete uniform
  frozen-build statistics from Gamma through 5x5x5;
- `data/dmc_ice13_gxtb_current_adaptive_set.csv`: densest qualified endpoint,
  error, adjacent change, and convergence state for every phase;
- `data/dmc_ice13_gxtb_current_adaptive_statistics.csv`: aggregate statistics
  for that current set, explicitly marked non-final;
- `data/dmc_ice13_gxtb_phasewise_progress.csv`: reproducible phase-wise
  progress snapshots at mesh limits 6x6x6, 7x7x7, and 8x8x8, including the
  selected mesh of every phase and every still-pending next endpoint;
- `data/dmc_ice13_gxtb_phase_vii_current.csv`: current Ice-VII sequence,
  including the qualified 7x7x7--8x8x8 adjacent pair;
- `data/dmc_ice13_published_dft_absolute_energies.csv`: published DMC/DFT
  reference data used for the comparison context;
- `data/geometries.json` and `poscars/`: canonical benchmark geometries.

progress values at mesh limits 6x6x6, 7x7x7, and 8x8x8 are 2.3596, 1.8643,

## Inputs and figures

- `gxtb_spglib_inputs/`: native-Bloch SPGLIB-reduced production inputs;
- `kpoint_inputs/`: earlier explicit full-grid templates retained only where
  useful for route comparison;
- `figures/dmc_ice13_gxtb_same_build_fixed_mesh_mae.*`: uniform same-build
  convergence plot generated by
  `scripts/make_dmc_same_build_mae_plot.py`;
- `reproduction/seidler_dmc13_recalculation/`: compact independent package
  with all primitive structures in absolute Cartesian coordinates, DMC
  references, available absolute energies by mesh, deterministic BvK
  generation, integrity records, and the qualified partial historical
  `mstore-inorganic` 4x4x4 matrix under `evidence/`.

The production method is native primitive-cell Bloch sampling with SPGLIB
reduction. Explicit Gamma supercells are validation oracles, not the production
route. Build identities and validation locations are documented in
[`../PART_I_PROVENANCE.md`](../PART_I_PROVENANCE.md).
